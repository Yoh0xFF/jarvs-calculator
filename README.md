## Description

Simple calculator to execute the math expression.

Supported operators ordered by precedence:

- Unary operators: `+`, `-`
- Factor operators: `*`, `/`
- Term operators: `+`, `-`

The calculator supports grouping expression as well.

Examples:

- `2.2 + 2.8`
- `5 * -2`
- `2 + 14 / 2`
- `(1 + 4) * 5 / (10 + -5)`
- `(2 + 7) * 2 + 2.5`

## NPM Commands

- Run the program: `npm run start`
- Run program in watch mode: `npm run start:dev`
- Run unit tests: `npm run test`
- Run unit tests in watch mode: `npm run test:dev`
